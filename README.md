# World Cup Odds Pro

Website demo dự đoán World Cup theo phong cách odds thể thao.

## Chức năng
- Giao diện tối hiện đại.
- Lịch thi đấu demo.
- Tỷ lệ 1X2, châu Á, tài/xỉu.
- Phiếu dự đoán bên phải.
- Lưu phiếu bằng Local Storage.
- Bảng xếp hạng demo.
- Responsive cho điện thoại.

## Cách đưa lên GitHub Pages
1. Tạo repository mới, ví dụ: `worldcup-odds-pro`.
2. Upload toàn bộ file trong thư mục này lên repository.
3. Vào Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main.
6. Folder: /(root).
7. Save.

Lưu ý: Đây là website mô phỏng dự đoán vui/nội bộ, không xử lý giao dịch cá cược tiền thật.
